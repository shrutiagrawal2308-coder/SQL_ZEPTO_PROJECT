# 🛒 SQL Zepto Project

##  Project Overview
This project analyzes the Zepto product dataset using MySQL. The objective is to perform SQL-based data analysis and generate meaningful business insights related to products, pricing, discounts, inventory, and categories.

##  Tools Used
- MySQL
- MySQL Workbench
- SQL

##  Dataset
The dataset contains information about Zepto products, including:
- Product Name
- Category
- MRP
- Discount Percentage
- Discounted Price
- Availability
- Quantity
- Weight (grams)
- Stock Status

##  SQL Analysis Performed
- Top 10 most expensive products
- Average discount by category
- Products with high MRP
- Inventory analysis by category
- Distinct product analysis
- Product categorization using CASE statements
- Sorting, filtering, and aggregation using SQL

##  Key Insights
- Identified high-value products based on MRP.
- Compared discounts across different product categories.
- Analyzed inventory availability and stock distribution.
- Applied SQL functions to generate business insights.
